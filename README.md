How to Run:
No installation needed. Just open catch-the-ball.html in a web browser like Chrome, Firefox, or Edge.

Built With:
HTML
CSS
JavaScript (Vanilla)
Falling Object Catch Game is a simple browser-based game built using plain HTML, CSS, and JavaScript. The idea is straightforward — balls fall from the top of the screen and the player must move a basket left and right to catch them before they hit the ground.
The game is controlled entirely with the keyboard arrow keys. Each caught ball adds one point to the score. The player starts with 3 lives, and every missed ball costs one life. When all lives are gone, the game ends and the final score is shown on screen.
